## Specifications

This is an addon for World of Warcraft that takes screenshots at various times.

Main types of screenshots:

- Level up
- Skill up
- Interval
- Death
- Login
- Logout

Features:

- Slash commands(/sb)
- Interface panel
- Types of screenshots
- Name screenshots
- Create folders for types of screenshots
